# 5_25 Fitness PWA

A mobile-first fitness PWA built with Next.js (App Router), Tailwind CSS, and Framer Motion.

## Setup

```bash
npm install
npm run dev
```

## Features

- **Zustand store** with persist middleware (LocalStorage) for workout state
- **Dawn palette** Tailwind theme (background, surface, citrus, teal)
- **Mobile-first layout** with safe area support
